Wyzwalacz RS-232

USB Thermometer

Pomoc

Wyzwalacz RS-232 umożliwia sterowanie urządzeniem podłączonym do portu szeregowego z poziomu programu USB Thermometer. Tym samym portem COM może sterować wiele różnych Wyzwalaczy. Dzięki temu możliwa jest konstrukcja rozbudowanego termostatu w wieloma źródłami pobudzeń.

Możliwe jest zarówno sterowanie pinami portu szeregowego jaki i wysyłanie zdefiniowanego strumienia bajtów. W celu ustawienia Wyzwalacza, oprócz podstawowych ustawień wspólnych dla wszystkich Wyzwalaczy, konieczne jest skonfigurowanie dodatkowych ustawień:
  • Com port Nazwa portu szeregowego. Dla systemu Windows jest to COMx, gdzie x jest numerem portu.
  • Baud rate Ilość bitów na sekundę. Określa prędkość transmisji portu szeregowego.
  • Bity Ilość bitów składających się na jeden bajt. Domyślnie jest to 8.
  • Bity stopu Ilość bitów a dokładniej odpowiednik czasu przesłania zadanej ilości bitów oddzielających transmisję jednego bajtu od drugiego.
  • Parzystość Dodaje bit (na końcu każdego bajtu) kontrolujący poprawność przesyłanych danych.
  • Pin RTS Możliwe są trzy ustawienia:
    • Do not change Stan pinu RTS nie zmieni się podczas wystąpienia wyzwolenia.
    • Set to '1' Pin RTS zostanie ustawiony w stan wysoki podczas wystąpienia wyzwolenia.
    • Set to '0' Pin RTS zostanie ustawiony w stan niski podczas wystąpienia wyzwolenia.
  • Pin DTR Możliwe są trzy ustawienia:
    • Do not change Stan pinu DTR nie zmieni się podczas wystąpienia wyzwolenia.
    • Set to '1' Pin DTR zostanie ustawiony w stan wysoki podczas wystąpienia wyzwolenia.
    • Set to '0' Pin DTR zostanie ustawiony w stan niski podczas wystąpienia wyzwolenia.
  • Wyślij komendę Włącza wysyłanie strumienia bajtów komendy podczas zdarzenia.
  • Komenda Bajty do wysłania. Możliwa jest edycja w dwóch trybach ASCI lub HEX. W trybie ASCI wprowadza się komendę używając znaków standardowych (dostępnych na klawiaturze: liter, liczb, znaków interpunkcyjnych). W trybie HEX można wprowadzić znak który nie jest dostępny na klawiaturze, wpisując jego wartość heksadecymalną.
