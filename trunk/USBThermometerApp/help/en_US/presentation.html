Data presentation formats

USB Thermometer

Help

USB Thermometer application has four data presentation formats of measured temperatures. There are as follows:
  • Graph Displays temperature changes in specified period of time.
  • Bar graph Displays actual temperature on a bar graph. Height of the bar match actual temperature level.
  • Table Displays a table contains measure results. Range of data is fully configurable.
  • Report Displays a report in HTML format. The report contains a graph of temperature changes, minimum, maximum, average value and linear trend.
