USB Thermometer

Help

Displays a graph of temperature changes in specified period of time. To add a sensor to the graph use drag and drop technique. Sensors are added automatically when the drop occurs. It is possible to add multiple sensors to one graph. To modify the graph use the tools form the toolbar of the graph window:
  •   Magnify Turns on a magnifier mode. Using the left mouse button select a rectangle of an area to magnify. To leave this mode pres the right mouse button.
  •   Expand Expands a visible area for 150% of current view.
  •   Adjust Adjusts a visible area of the graph to minimum and maximum visible temperatures.
  •   Delete a sensor Removes a sensor from the graph.
  •   Cursors Turns on cursors mode. After point of a data point two cursors will appear. It will be possible to read a creation time of the sample and its value. Additionally the creation time and value will appear on the status bar of the graph window. To leave this mode press the right mouse button.
  •   3min - 3mth Automaticaly sets up visible data range to the specified period. Current date is set as a last visible date.
