Configuration

USB Thermometer

Help

The configuration windows consists of five parts:

General

It allows to configure basic settings of the application.
  • Start with starting Windows Selecting of this option causes adding an entry to the system registry with starts the application when the operating system starts. Not selecting of this option removes the entry.
  • Start minimized If selected Show icon in tray, it adds an icon to the system tray when the application starts. In other case it runs the application in minimized state.
  • Show icon in tray Adds an icon to the system tray.
  • Permit only single instance of the application If this option has been selected and the application already is running, when a user is trying to start it again, the application will not start. The user will be switched to the first instance of the application. In other case the second instance of the application will run normally.
  • Minimize to tray on minimize If this option and Show icon in tray are selected, the application main window will be hidden when the minimize button is pressed. In other case it minimizes normally.
  • Minimize to tray on close If this option and Show icon in tray are selected, it ignores the close button of the main window. When the close button has been pressed, it only hides the window (not close it). In that case choose Close option from the menu bar to close the application.
  • Temperature unit It is possible to choose one of the two temperature units: Celsius or Fahrenheit.
  • Sampling period The parameter configures the sampling period. It is possible to set the Sampling period with accuracy of 1s. Notification!!! If a number of connected sensors to the one USB Thermometer device is large, it is possible the sampling period will be expanded to the time of reading temperature from all sensors.

Visual

Configures preferred colors of Graphs.
  • Background color The color of the graph background.
  • Foreground color The color of the graph grid, values and cursor
  • Trace color Those are preferred colors of data sets added to the Graph. Data sets added one by one to the graph will assume the selected trace colors.

Database

Settings and the command parser of the database.
  • Database file Selects the database file where the application data are stored. This option allows to change the path to the database file.
  • Command parser It allows to run the sqlite3 commands. More information about the database you can find in the chapter Database.

Language

On this page it is possible to change a localization of the application.

Connections

Configures network connections run by the application when the TCP/IP and UDP triggers occurs. Those settings allows to connect through a proxy server.
  • Host name or IP address Network address of the proxy server.
  • Port Port used to connect to the proxy server.
