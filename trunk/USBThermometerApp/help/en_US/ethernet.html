Help

TCP/IP and UDP trigger allows to send a string of bytes (command) to a remote computer or device using specified internet protocol. To set up the trigger settings, aside from setting up the basic configuration (common for all types of the triggers, it is necessary to configure additional settings:
  • Host Network name or IP address of the remote computer or device.
  • Port Port number of the remote computer or device.
  • Send a command Bytes to send. It is possible to edit message using two modes: ASCI or HEX. In ASCI mode command is typed using standard characters which are available on keyboard (letters, numbers, etc). In HEX mode it is possible to type any byte value by typing its hexadecimal value.
