USB Thermometer

Help

Displays actual temperature on a bar graph. Height of the bar match actual temperature level. It is possible to display only one temperature in one bar graph window. It is not possible to regulate of a visible range. The bar graph has unique functionality. It is possible to detach the bar graph window from the main window. To do so use the following icon . The detached window stays always on top also when the main window has been minimized. To return to the normal state of the bar graph window use another button (with appears after the window has been detached) .
