USB Thermometer

Help

Displays a report in HTML format. The report contains a graph of temperature changes, minimum, maximum, average value and linear trend. To add a sensor to the report use drag and drop technique. Sensors are added automatically when the drop occurs. It is possible to add multiple sensors to one report. To modify the report use the tools form the toolbar of the report window:
  •   Date and time configuration Opens the report configuration window. In this window it is possible to configure display period of time.
  •   Report save Saves the report to a HTML file.
  •   Refresh Refresh the report content.
  •   Delete a sensor Removes a sensor from the table.
