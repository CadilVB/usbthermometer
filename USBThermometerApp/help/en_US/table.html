USB Thermometer

Help

Displays a table contains values of measured temperatures. To add a sensor to the table use drag and drop technique. Sensors are added automatically when the drop occurs. It is possible to add multiple sensors to one table. To modify the table use the tools form the toolbar of the table window:
  •   Date and time configuration Dispalys a table configuration window. There are three possible display modes:
    • Last saved Displays constant number of samples last added to the database. In this mode remaining parameters are disabled.
    • Period of time Displays measured temperature values in given period of time.
    • Last It is similar mode to the Period of time, but it displays measured temperature values made in period of time relative to current date and time.
    Visible samples may be filtered. It is possible to configure visible values range. In default maximum possible values range is set. It means all samples will be displayed. It is also possible to sort samples from the oldest to the newest and from the newest to the oldest. Set up the total limit of visible rows is necessary due to limit search time in the database.

  •   Table save Saves the visible table to an Excel file.
  •   Refresh Refresh the table content. It is useful when you use the modes Last saved and Last. Data in tables is not refreshed automatically as it happens in the Graphs and the Bar graphs.
  •   Delete a sensor Removes a sensor from the table.
