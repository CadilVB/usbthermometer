Help

User trigger executes defined external programs. To set up the trigger settings, aside from setting up the basic configuration (common for all types of the triggers, it is necessary to configure additional settings:
  • Shell command Command which executes a program. Type full path to an external application (unless the program path is included in the system variable called PATH) and its parameters.
An example command runs the msg.exe program, which displays a message box on the screen:
msg.exe * %computername% Overheated!
